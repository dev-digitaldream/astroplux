A small checklist I use to keep Lighthouse scores high:

- Ship only what you need (avoid heavy client JS).
- Prefer static over dynamic where possible.
- Optimize images and use responsive sizes.
- Inline critical CSS when it makes sense.
- Cache aggressively at the edge/CDN.
